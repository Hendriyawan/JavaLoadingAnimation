# JavaLoadingAnimation
(Java) simple to display loading animation

```
Loading wave = new Loading(true, "Loading message...", LoadingType.WAVE);
wave.start();
//OR
new Loading(true, "Loading message...", LoadingType.WAVE).start();
```
# LoadingType
```
LoadingType.WAVE
```
```
LoadingType.SPINNER
```
